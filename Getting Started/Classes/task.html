Aufgabe:

  1. Erstelle eine Data Class namens Address, welche (in der richtigen Reihenfolge) die finalen Parameter:
    • street vom Typ String
    • houseNumber vom Typ String
    • postalCode vom Typ Int
    • city vom Typ String
    im Konstruktor entgegen nimmt.
  2. Stelle sicher, dass setter & getter entsprechend der Veränderlichkeit zur Verfügung gestellt werden.
  3. Erstelle eine Klasse AddressBook, welche eine finale veränderliche Liste von Adressen im Konstruktor entgegen nimmt und passende setter & getter zur Verfügung stellt
  4. Vervollständige die Funktion fun contains(address: Address) der Klasse AddressBook so, dass sie ein Boolean zurück gibt, welches angibt, ob das AddressBook die eingegebene Adresse oder eine Addresse mit den gleichen Werten enthält.
